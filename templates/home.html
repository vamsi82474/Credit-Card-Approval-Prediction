Credit Card Approval Prediction

An end-to-end Machine Learning system that detects fraudulent transactions and predicts credit card approval using advanced classification algorithms.

🚀 Make a Prediction

About the Project

📊

Dataset

Uses the Kaggle Credit Card Fraud Detection dataset containing 284,807 transactions with 30 PCA-transformed features and a binary fraud label.

🤖

Models

Trains and compares four classifiers: Logistic Regression, Decision Tree, Random Forest, and Gradient Boosting. The best-performing model is saved and deployed.

🌐

Deployment

A lightweight Flask web application allows bank staff to enter transaction details and receive instant approval or rejection decisions.
